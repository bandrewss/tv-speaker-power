# TV Speaker Power
An Arduino listens for the TV remote power signal and flips a relay.

### Wiring
3 legged IR sensor
- Ground
- 5V
- Signal Pin 13

### Red/Green LED
- Ground through resistor
- Red to Pin 4
- Green to Pin 5

### Relay
- Ground
- Toggle to Pin 6

### Override Button
- Between 5V and Pin 2
- Pin 2 tied to ground through resistor
